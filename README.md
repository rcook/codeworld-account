# codeworld-account by Richard Cook

Haskell application with main executable only built using Stack

## Clone repository

```
git clone https://github.com/rcook/codeworld-account.git
```

## Install compiler

```
stack setup
```

## Build

```
stack build --fast
```

## Run application

```
stack exec codeworld-account
```

## Run tests

```
stack test
```
